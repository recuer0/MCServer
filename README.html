INSTALAR MINECRAFT SERVER

Buscar "Minecraft server" en internet

https://www.minecraft.net/es-es/download/server

Descargar minecraft server

Copiar el comando que indique en la web y ponerlo en la misma carpeta que lo instalaste

C:\Users\mclovin\Desktop\new-server\server> java -Xmx4G -Xms4G -jar server.jar nogui

Asegúrate de que ponga server.jar y se encuentre dentro de una carpeta llamada "server"

Dentro de la carpeta "mine-server" deben haber 2 carpetas, "dashboard" y "server"

Tu carpeta debe quedar así

Aceptar EULA - (términos y conddiciones)

En eula.txt cambia:

false → true

Iniciar servidor

Ejecuta el mismo comando otra vez para iniciar el servidor

Estructura final

mine-server
│
├── server
│   ├── libraries/
│   ├── logs/
│   ├── versions/
│   ├── world/
│   ├── banned-ips.json
│   ├── banned-players.json
│   ├── eula.txt
│   ├── ops.json
│   ├── server.jar
│   ├── server.properties
│   ├── usercache.json
│   └── whitelist.json
│
└── dashboard
    ├── app.js
    ├── index.html
    ├── style.css
    ├── sidebar.css
    ├── main.js
    ├── preload.js
    ├── package.json
    ├── package-lock.json
    └── node_modules/

Instalar y ejecutar en la carpeta "dashboard"

npm install
npm start

Para abrir el servidor en minecraft
